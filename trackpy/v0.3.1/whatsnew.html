What’s new in the package¶

A catalog of new features, improvements, and bug-fixes in each release. Follow links to the relevant GitHub issue or pull request for specific code changes and any related discussion.

v0.3.1¶

Enhancements¶

  • Added pair correlation functions in 2D and 3D (GH#336)
  • Mean squared displacement calculations are up to 40 times faster due to a new FFT-based algorithm (GH#337)
  • Number of measurements in MSD calculations is more accurate (GH#337)
  • Increased performance of compute_drift (GH#345)

Bug Fixes¶

  • Bug in the python refinement code was solved: feature finding with engine=’python’ is now more accurate. (GH#377)
  • Error in subtract_drift is solved (GH#351)
  • Legends are disabled by default in plotting (GH#357)
  • Plots in (y, x) now always have their y axis inverted (GH#357)
  • Empty frames are not ignored anymore in link_df (GH#293)
  • characterize was not passed to refine in locate (GH#310)
  • annotate3d now works with the notebook backend (GH#308)
  • Trackpy is now fully compatible with pandas 0.17 and 0.18.1
  • Trackpy is compatible with Pillow >0.3.0 (through PIMS v0.3.3)
  • The pos_columns default argument is not mutable anymore in msd, imsd, emsd, and compute_drift. (GH#337)

API changes¶

  • max_lagtime in msd, imsd, and emsd is now included in the returned lagtimes (GH#337)
  • subtract_drift does not act inplace by default (GH#351)
